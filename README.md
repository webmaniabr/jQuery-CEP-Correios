# ⚠️ Acesso Público Descontinuado

> **A partir de 1º de Abril de 2026, o acesso gratuito e público da API de CEP será desabilitado.** Todas as requisições deverão obrigatoriamente incluir as credenciais de autenticação na header da requisição.

**Obtenha suas credenciais de acesso:** [Fale Conosco](https://webmaniabr.com/chat/)

---

## Por que o acesso público está sendo descontinuado?

O acesso público e sem autenticação apresenta riscos de segurança, compromete a estabilidade da infraestrutura e reduz a qualidade do serviço oferecido. Para garantir uma experiência confiável e de alto nível, a API de CEP passará a estar disponível exclusivamente para clientes que contratarem as nossas soluções. Continuamos oferecendo as mesmas funcionalidades e qualidade que você já conhece.

## O que muda na prática?

A partir de **1º de Abril de 2026**, requisições realizadas sem as credenciais `app_key` e `app_secret` na header serão rejeitadas. Para continuar utilizando a API de CEP:

1. **Solicite as suas credenciais** com o nosso setor comercial: [webmania.com.br/chat](https://webmania.com.br/chat/)
2. **Inclua as credenciais** em todas as requisições conforme a [documentação](https://webmania.com.br/docs/rest-api-correios/#autenticacao)
3. **Atualize suas integrações** antes da data limite para evitar interrupções
